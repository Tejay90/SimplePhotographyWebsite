# LENS — Photography Portfolio

A high-performance, responsive portfolio website built with JavaScript, HTML5, and CSS3. Developed in Visual Studio Code, this project showcases a modern photography platform featuring seamless SPA navigation, interactive galleries, and advanced form validation without relying on external frameworks or libraries.

## Key Features

- Instant page transitions between Portfolio, Landscape, Contact, and Impressum sections using the History API and dynamic DOM manipulation.
- Custom-built image viewer with smooth zoom, keyboard navigation (Escape key), and category filtering.
- A fully functional burger menu with scroll locking and smooth animations on mobile devices.
- Client-side validation with visual feedback (shake animation for errors, success message on submit) and email format checking.
- Elements fade and slide into view as the user scrolls using the modern IntersectionObserver API for optimal performance.
- The navbar background opacity changes dynamically on scroll for improved readability.

## Technologies & Tools

- JavaScript: Core logic, event delegation, DOM manipulation, and state management.
- HTML5: Semantic structure for accessibility and SEO.
- CSS3: Custom properties (`:root`), Flexbox, CSS Grid, animations, transitions, and responsive design.
- Fonts: Google Fonts (Cormorant Garamond, Syne) for typography.
- Development Environment: Visual Studio Code.

# LENS — Fotografie-Portfolio

Eine responsive Portfolio-Website, entwickelt mit JavaScript, HTML5 und CSS3. Erstellt in Visual Studio Code, zeigt dieses Projekt eine moderne Plattform für Fotografie mit nahtloser SPA-Navigation (Single Page Application), interaktiven Galerien und fortschrittlicher Formularvalidierung, ohne auf externe Frameworks oder Bibliotheken zu setzen.

## Hauptfunktionen

- Sofortiger Wechsel zwischen den Abschnitten Portfolio, Landschaft, Kontakt und Impressum unter Nutzung der History API und dynamischer DOM-Manipulation.
- Maßgeschneiderter Bildbetrachter mit sanftem Zoom, Tastaturnavigation (Escape-Taste) und Kategorien-Filterung.
- Voll funktionsfähiges Burger-Menü mit Scroll-Locking und sanften Animationen auf mobilen Geräten.
- Elemente blenden und gleiten beim Scrollen sanft ein, basierend auf der modernen IntersectionObserver API für optimale Performance.
- Der Hintergrund der Navigationsleiste ändert seine Deckkraft automatisch beim Scrollen für bessere Lesbarkeit.

## Technologien & Tools

- JavaScript: Kernlogik, Event-Delegation, DOM-Manipulation und State-Management.
- HTML5: Semantische Struktur für Barrierefreiheit und SEO.
- CSS3: Eigene Eigenschaften (`:root`), Flexbox, CSS Grid, Animationen, Übergänge und responsives Design.
- Schriftarten: Google Fonts (Cormorant Garamond, Syne) für typografisches Design.
- Entwicklungsumgebung: Visual Studio Code.
